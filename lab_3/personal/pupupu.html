История создания языков программирования

Python создал в 1989-1991 годах голландский программист Гвидо ван Россум.

C разработал в 1969-1973 годах сотрудник Bell Labs Деннис Ритч.

Java разработал в 1990-1996 годах канадский программист Джеймс Гослинг.

JavaScript разработан в 1996 году, автор - американский программист Брендан Айк.

Prolog разработан в 1972 году, автор - французский программист Ален Колмероэ.

Ruby разработан в 1993-1995 годах, автор - японский программист Юкихиро Мацумото.
